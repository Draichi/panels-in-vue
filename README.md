# panels

![screenshot](usage.gif?raw=true "screenshot")


> Panels in list with vue.js

## Start API
> On `api` folder
```bash
# install
npm install

# start database (if you are on linux)
sudo service mongod start

# start database (if you are on macOS)
sudo mongod

# start database (if you are on Windows as administrator)
"C:\Program Files\MongoDB\Server\3.4\bin\mongod.exe"
"C:\Program Files\MongoDB\Server\3.4\bin\mongo.exe"

# start server
npm start
```

## Build Setup
> On `panels-in-vue` folder
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
